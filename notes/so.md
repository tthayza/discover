<h1>Sistema Operacional</h1>

<p>Sistema operacional é a interface entre usuário e máquina. Programa que irá controlar certo aparelho. </p>

<p>Possui características como facilidade no entendimento para o usuário, controle de hardware e gerenciamento de softwares. </p>

<h2><b>Tipos de interfaces do usuário</b></h2>

**GUI Graphical User Interface**

- Interface gráfica, elementos gráficos.

**CLI Command-Line Interface**

- Linha de comando, emite comandos em texto para o computador.

<hr>

<h2><b>Módulos Operacionais</b></h2>

**<h3>Kernel</h3>**

- Também conhecido como núcleo, é responsável por gerenciar o processador,memória, armazenamento e dispostivos.

**<h3>Gerenciador de Processos</h3>**

- processo é um programa em execução;
- agendamento de processos;

- **<h3>Thread</h3>**
  - divisão de um processo para melhor performance;
  - uma parte do processo;
  - poderá executar em paralelo;

**<h3>Multitarefa (multitasking)</h3>**

- Muitas tarefas que podem ser executadas simultaneamente;
- Troca muito rápida entre elas;

**<h3>Gerenciador de Arquivos (file System)</h3>**

- Responsável pela organização e armazenamento dos arquivos;
